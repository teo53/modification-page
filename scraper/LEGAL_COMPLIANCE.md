# 법적 준수 가이드라인 (Legal Compliance Guidelines)

이 문서는 웹 스크래핑 시 고려해야 할 법적, 윤리적 사항을 설명합니다.

## 1. 서비스 약관 (Terms of Service)

### 확인 사항
웹 스크래핑 전에 반드시 대상 웹사이트의 서비스 약관을 확인하세요:

- **이용약관 페이지** 확인
- **robots.txt** 파일 확인 (`https://example.com/robots.txt`)
- **API 제공 여부** 확인 (공식 API가 있다면 스크래핑 대신 API 사용 권장)

### robots.txt 준수

```
User-agent: *
Disallow: /admin/
Disallow: /private/
Crawl-delay: 10
```

위와 같은 robots.txt가 있다면:
- `/admin/`, `/private/` 경로는 스크래핑 금지
- 요청 간 최소 10초 간격 유지

### 일반적 제한사항

많은 웹사이트들이 다음을 금지합니다:
- 자동화된 대량 데이터 수집
- 수집한 데이터의 상업적 사용
- 수집한 데이터의 재배포
- 서버에 과도한 부하를 주는 행위

## 2. 개인정보보호법 (Privacy Laws)

### 대한민국 관련 법규

1. **개인정보 보호법**
   - 개인정보 수집 시 정보주체의 동의 필요
   - 수집 목적 외 이용 금지
   - 개인정보 안전성 확보 의무

2. **정보통신망 이용촉진 및 정보보호 등에 관한 법률**
   - 정보통신망을 통한 개인정보 수집 규제
   - 불법 정보 유통 금지

### 수집 시 주의사항

다음 정보는 특히 주의가 필요합니다:
- 이름, 전화번호, 이메일 주소
- 주민등록번호, 여권번호 등 고유식별정보
- 건강정보, 성적 지향, 범죄경력 등 민감정보
- 위치정보

### 권장사항

1. **최소 수집 원칙**: 필요한 정보만 수집
2. **익명화/가명화**: 개인 식별이 불가능하도록 처리
3. **보관 기간 제한**: 수집 목적 달성 후 즉시 파기
4. **접근 제한**: 수집된 데이터에 대한 접근 권한 관리

## 3. 저작권법 (Copyright Law)

### 보호 대상

웹사이트의 다음 요소들은 저작권으로 보호될 수 있습니다:
- 텍스트 콘텐츠 (기사, 설명문 등)
- 이미지, 사진, 동영상
- 데이터베이스 구조 및 내용
- 웹사이트 디자인

### 공정 이용 (Fair Use)

다음 목적의 사용은 공정 이용으로 인정될 수 있습니다:
- 비평, 논평, 보도
- 교육, 학술 연구
- 비영리적 개인 사용

**주의**: 공정 이용의 범위는 사안별로 다르므로 법적 자문을 구하세요.

### 대한민국

- **정보통신망법**: 정보통신망에 대한 무단 침입 금지
- **형법**: 컴퓨터등장애업무방해죄

### 주의사항

다음 행위는 범죄에 해당할 수 있습니다:

- 서버 과부하를 유발하는 행위 (DoS)
- 보안 취약점 악용
- 유료 콘텐츠 무단 접근

## 5. 윤리적 스크래핑 가이드라인

### Do's (해야 할 것)

1. **적절한 요청 간격 유지**
   ```
   최소 1-2초 이상의 지연 시간 설정
   서버 응답 시간에 따라 동적 조절
   ```

2. **User-Agent 명시**
   ```
   봇임을 명확히 표시하고 연락처 포함
   예: "MyBot/1.0 (contact: admin@example.com)"
   ```

3. **오류 처리**
   ```
   429 (Too Many Requests) 응답 시 즉시 중단
   5xx 서버 오류 시 재시도 제한
   ```

4. **캐싱 활용**
   ```
   이미 수집한 데이터 재요청 방지
   변경된 페이지만 선별적 수집
   ```

### Don'ts (하지 말아야 할 것)

1. **과도한 요청 금지**
   - 초당 수십 회 이상의 요청
   - 병렬 요청으로 서버 과부하

2. **보안 우회 금지**
   - CAPTCHA 자동 우회
   - 로그인 세션 탈취
   - IP 차단 우회 (프록시 악용)

3. **민감 데이터 수집 금지**
   - 비공개 정보
   - 개인 식별 정보
   - 유료 콘텐츠
