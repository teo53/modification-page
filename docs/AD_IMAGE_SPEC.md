# 광고 이미지 규격 가이드 (AD_IMAGE_SPEC.md)

## 📐 통일 이미지 규격

### 표준 광고 이미지
| 규격 | 값 |
|------|-----|
| **크기** | 258 × 138 px |
| **비율** | 43:23 |
| **포맷** | JPG, PNG, GIF |
| **최대 용량** | 500KB |

> 레퍼런스: queenalba.net 광고 이미지 규격

## 🖼️ 광고 상품별 이미지 사용

### 1. 프리미엄채용정보 (PremiumJobGrid)
- **규격**: 258 × 138 px
- **사용처**: 메인 이미지 카드
- **특징**: 컬러 테두리 (핑크/퍼플/블루)

### 2. 우대채용정보 (CompactAdGrid)
- **규격**: 48 × 48 px (로고)
- **사용처**: 정사각형 업체 로고
- **특징**: 텍스트 정보 위주

### 3. VIP/Special 광고 (PremiumAdGrid/SpecialAdGrid)
- **규격**: 300 × 400 px (3:4 비율)
- **사용처**: 썸네일 카드
- **특징**: 세로형 카드 레이아웃
